Typh

Introduction


Typh is students database management software. It is based on a client-server model where each user installs a client application on their system.
The client connects to the server which is also installed into the network itself. Typh is meant for private networks only.
The required softwares are installed in a server node to which all the clients connect.

System Requirements

Typh can run only on Desktop or Laptop systems.
Following are the minimum hardware requirements:

Processor
- 2.4 GHz CPU
Graphics
- Minimum 256 MB
Memory
- 2 GB.
Storage
  • For Server: 1 GB.
  • For Client: 5 MB.
Following are the software requirements:
Operating System
  1. Windows 7 64 bit or Later.
  2. Any Linux distribution.
Java
- Java 8 or Later

Features

Client User :-

Login

  • The Login username and password is provided by the administrator.
  • Upon Login, the user's last login, total students, etc are provided in the side bar.

Students Profile

  • The user can view academic data of any student but can edit only of those who the user is assigned to i.e. class teacher.
  • Students data is divided into following sections:
    1. Personal
    2. Academic
    3. Attendance
    4. Projects
    5. Assignments
  • Use Image size of 128x128 only.
  • The user can also export the data in Excel format.
  • The user has the use update button to update changes to the database.

Admin User :-

The admin controls and manages all the other accounts including student's profiles. The admin can perform the following actions:

  • Create or Delete User Accounts
  • Create or Delete Student's Base profile
  • Send short text message to any client who is online
